# JWCx Countdown Timer
เขียนด้วยความเร็วแสง ไม่ได้ clean code

## Features
- Start / Stop
- Pause / Resume
- ปิด Tab แล้วเปิดใหม่ เวลาก็ยังเดินอยู่
- No server-side
- Work 100% offline (clone first)
